# Protocol 12: Ultra-Dark Cyberpunk Theme Overhaul

## 1. Objective

Transform the portfolio into a high-density, "Dark Cyberpunk" environment with deep contrast, neon accents, and gritty technical textures.

## 2. Global Style Updates (`globals.css`)

- **Deep Space Palette**:
  - Background: Nearly black (`#02040c`) with a subtle radial gradient.
  - Card: Deep navy-black (`#050814`).
- **Neon Accents**:
  - Primary: Electric Blue (`#3c8cf` / Cyan shift).
  - Secondary: Crimson Magenta (`#ff0080`).
  - Accent: Pure Neon Cyan (`#00fff0`).
- **Texture & FX**:
  - Added a global **Noise Overlay** (via `cyber-grid::after`) to provide a grainy, film-noir cyberpunk feel.
  - Enhanced **Glitch Hover** effects with RGB split text shadows.
  - Darkened **Glass-morphism** layers for higher text legibility against dark backgrounds.

## 3. Component Enhancements

### Hero Section

- **Intense Atmosphere**: Large blurred ambient glows in the background.
- **Tactical Buttons**: Skewed, high-contrast buttons with intense outer glows.
- **Cyborg Imaging**: Enhanced the image unit with scanlines, HUD overlays, and chromatic aberration on hover.

### Technical Array (Skills)

- **Neon Coding**: Replaced standard tailwind colors with system-defined primary/secondary/accent gradients.
- **Subsystem Status**: Added "Optimized," "Stable," and "Encrypted" status indicators to each module.

### Deployment Log (Experience)

- **Node Reactor**: Added pulsing neon nodes on the timeline axis.
- **Tactical Payload**: Redesigned technical badges to be more compact and technical.

### Projects Section

- **Execution Modules**: The grid layout now sits on a deeper background, making the neon borders and scanlines pop significantly more.

## 4. Visual Integrity

- **Typography**: Heavy reliance on monospace fonts for technical data and ultra-bold black weights for headers.
- **Interactivity**: Every interactive element now features a "Cyber Pulse" or "Glitch" response.
